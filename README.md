# newsAppNew

Реализовано: 
 - Добавление в избранное
 - Удаление из избранных
 - Отображение избранные в отдельном окне, используя Realm
 - Поиск по избранным новостям
 - Подгрузка новостей по листанию, до 7-ми дней
 - Обновить ленту (pull-to-refresh)

НЕ реализовано:
 - "Show more" кнопка
